# ezcam

EZCAM은 AI 기반 웹캠 배경 제거 및 크로마키 기능을 제공하는 데스크탑 앱입니다.
실시간으로 배경을 제거하고, 초록색 배경 합성을 통해 크로마키 윤곽을 확인할 수 있습니다.
주로 인터넷 방송인(스트리머)이 캠 화면을 오버레이할 때 활용할 수 있습니다.

## 주요 기능

- AI(RVM) 기반 실시간 배경 제거
- 초록색 배경 합성으로 크로마키 확인 가능
- 카메라 선택 및 실시간 영상 표시

## 설치 및 실행

1. 파이썬 설치
2. 의존성 설치 : `pip install -r requirements.txt`
3. AI 모델 가중치 파일 원본 링크 : https://github.com/PeterL1n/RobustVideoMatting/tree/master
